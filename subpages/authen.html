What is


Authentication serves as a means to "prove that I am who I say I am."

Authentication means to establish trust between two parties, so that data can be safely shared.
    Ways to Authenticate:
    • Provide something you know - a password
    • Provide something you have - a SmartCard or Code
    • Provide something you are - a fingerprint (biometrics)

    Multi Factor Authentication (MFA):
    • Has the user provide multiple things before granting access to an account, service, etc.
    • Adds multiple layers of security
    • Could help compensate for a weak password
    • Common MFA methods include Email/SMS Tokens and Security Questions
